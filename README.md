# Homez Project


## Description

Real estate platform for listing and exploring properties efficiently.

## Technologies Used 
- React
- React Router
- TypeScript
- Axios (for API calls)
- React Query
- Vite.js
- Tailwind CSS
- JSON Server
- Swiper
- React-Leaflet
- React Icons
- AOS
- ESLint

## Installation
To start the Front-end of the application, run the following command: 
- npm run dev

To run the Back-end of the application, run the following command: 
- npm run start
